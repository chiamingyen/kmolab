2016Fall 的計算機程式課程已經從 Github 的應用開始, 先讓學員利用 Git 分散式管理工具進行團隊間的數位資料管理, 然後再以線上的分組程式為例, 說明運用計算機程式, 如何將原本繁雜的資料排序流程, 導向能隨需求而快速因應變化的網際程式流程.
